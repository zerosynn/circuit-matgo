# Circuit Matgo

전자회로 기판 콘셉트의 화투 이미지로 만든 브라우저용 1인 맞고 게임입니다. 플레이어와 CPU가 한 판씩 겨루며, 데스크톱과 모바일 화면을 모두 지원합니다.

## 주요 기능

- 48장 전체 화투 이미지 사용
- 플레이어 10장, CPU 10장, 바닥 8장의 맞고 배분
- 같은 월 패 맞추기와 바닥패 선택
- 광, 열끗, 띠, 피 점수 계산
- 오광, 사광, 삼광, 비삼광, 고도리, 홍단, 초단, 청단
- 고 / 스톱과 3고 이상 배수
- 광박, 피박, 멍따, 흔들기, 폭탄, 쓸
- CPU 자동 플레이
- 기기별 승패 및 연승 기록
- 모바일 반응형 UI

## 로컬 실행

Node.js 22.13 이상이 필요합니다.

~~~bash
npm install
npm run dev
~~~

브라우저에서 터미널에 표시된 로컬 주소로 접속합니다.

## 배포 빌드

~~~bash
npm run build
npm run start
~~~

GitHub 저장소에 올린 뒤 Cloudflare Workers, Vercel 등 Node/Worker 기반 배포 서비스에 연결할 수 있습니다.

## 프로젝트 구조

- `app/page.tsx`: 게임 화면과 사용자 상호작용
- `app/game.ts`: 카드 구성, CPU, 맞고 점수와 진행 규칙
- `app/globals.css`: 디지털 회로 테이블 디자인과 반응형 UI
- `public/cards`: 웹용으로 최적화한 화투 48장

## 규칙 범위

지역과 서비스마다 다른 맞고 세부 규칙 중 대중적인 2인 맞고 기준을 적용했습니다. 첫 스톱 기준은 7점이며, 게임은 현금이나 결제를 사용하지 않는 점수제입니다.

## 이미지 권리

`public/cards`의 화투 이미지는 이 프로젝트 제작자가 만든 원본 디자인을 웹 게임용으로 분리하고 최적화한 자산입니다.
